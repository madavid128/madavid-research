Source: tools/gallery_master.csv
Tip: edit the CSV in a spreadsheet, then rebuild YAML and watermarks.
Column reference (what each column can do)
  • collection: pictures or art
  • image: filename or images/<name>; web-friendly ext preferred (.jpg/.png)
  • tags: semicolon-separated (tag1; tag2; tag3)
  • style: square or banner (controls aspect ratio in the grid)
  • subtitle / alt: optional text shown in the lightbox and for accessibility
  • featured: mark as a gallery highlight card
  • featured_rank: ordering for highlights (lower shows first)
  • home_slot: selects the Home page highlight image: publications, projects, team, art
  • project_area: selects the main Projects page card image: cartilage, tendon, imaging-ml, other
  • project_area_rank: priority when multiple rows share the same project_area (lower wins)
  • project_page_section: attaches images to a project subpage without affecting the main Projects cards (same values as project_area)
  • project_page_rank: ordering within a project subpage image gallery (lower shows earlier)
  • header_background: true/false to include in the rotating Home header background
  • header_background_rank: ordering for header rotation (lower shows earlier)
  • share_page: picks an image to be used as the social preview (og:image) for a page: home, publications, projects, team, art, pictures, updates
  • share_page_rank: priority when multiple rows share the same share_page (lower wins)
